# Five-Star Feedback

**Kategori:** Tilgangskontroll
**Vanskelighetsgrad:** ⭐⭐
**Mål:** Slett alle femstjerners tilbakemeldinger.

## Hint

Hintene gir gradvis mer hjelp. Prøv hint 1 først, og gå videre bare om du står fast.

> **Merk:** Du må være logget inn som admin. Løs Login Admin og Admin Section først.

### Hint 1
Denne er lett om du allerede har løst Admin Section. Gå til admin-siden og finn tilbakemeldingene.

### Hint 2
Slett alle tilbakemeldinger med fem stjerner ved å klikke på søppelbøtte-ikonet. Når alle femstjerners er borte, løses oppgaven.
